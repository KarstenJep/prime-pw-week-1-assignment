# 8 - Why would you run the command 'git status'?
It will show you the status of git in a specific folder. This will identify any untracked files, or changes to be committed.

# 9 - What does the command 'git add .' do?
It tells the terminal to add everything in that specific folder to git (the precursor to staging a commit)
